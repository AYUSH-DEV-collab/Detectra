# 🛡️ DETECTRA: The Neural Defense Grid

**Detect. Analyze. Defend.**  
Detectra is a high-fidelity front-end prototype designed to identify and map synthetic media artifacts. In an era where reality is programmable, Detectra stands as the final line of defense against deepfakes, leveraging neural artifact mapping to ensure digital integrity.

---

## 🚀 Key Features

- **Interactive Neural Scanner:** A simulated forensic upload system that analyzes media for GAN-specific fingerprints and temporal inconsistencies.
- **Deep-Layer Pattern Mapping:** Visual representation of how synthetic models leave invisible neural "sweat" on digital media.
- **Real-Time Simulation Logic:** Built-in JavaScript engines (`setInterval`/`setTimeout`) to simulate complex AI analysis workflows.
- **Cyber-Forensic Aesthetic:** A premium "Dark-Neon" UI/UX featuring glassmorphism, smooth page transitions, and a monospaced technical design.
- **Semantic Navigation:** Fully accessible routing using React Router 7 with active state tracking.

---

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS3 + [TailwindCSS 4](https://tailwindcss.com/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Icons/Aesthetics:** Custom SVG paths & Space Grotesk Typography

## 📦 Installation & Setup

To run the Neural Grid locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AYUSH-DEV-collab/Detectra.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Detectra
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Launch the development terminal:**
   ```bash
   npm run dev
   ```

5. **Access the interface:** Open `http://localhost:5173` in your browser.

---

## 🧪 Simulation Flow 

The project currently operates as a **Frontend Prototype**. The analysis logic is simulated to showcase the intended user experience:

- **Ingestion:** Uses `useRef` to trigger a hidden forensic file input.
- **Processing:** Uses `useState` and `setInterval` to drive a 100% linear progress bar.
- **Result Generation:** Uses a randomized state generator to produce mock forensic reports (e.g., `HIGH RISK`, `AUTHENTIC`, or `ANOMALY`).
